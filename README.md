# Banking Intelligence: Dashboard Itau & Financial Analysis

## Overview
Este repositório apresenta uma prova de conceito focada no setor bancário, analisando dados do Itaú Unibanco. O objetivo é demonstrar como a integração entre Python e SQL permite extrair insights estratégicos de balanços complexos, otimizando o tempo de análise e garantindo a integridade das informações extraídas.

## Tech Stack e Funcionalidades
* **Processamento de Dados:** Python (Pandas/Numpy) para saneamento de bases do BACEN e CVM.
* **Visualização:** Estruturação de indicadores de performance bancária como ROE, Índice de Basileia e Margem Financeira.
* **Banco de Dados:** Consultas estruturadas para manipulação de grandes volumes de dados financeiros.
